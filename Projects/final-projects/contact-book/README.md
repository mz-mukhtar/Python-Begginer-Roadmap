# Capstone Track 2: Contact Book

## 📌 Project Description
Build a multi-category personal and professional contact registry featuring JSON persistence, category filtering, and CSV export.

## 🎯 Requirements
- Menu: 1. Add Contact, 2. View Contacts, 3. Search Contact, 4. Update Contact, 5. Delete Contact, 6. Export Contacts, 7. Exit
- Fictional contact records (`CON-001`, Name, Phone, Email, Category)
- Automated unit tests
