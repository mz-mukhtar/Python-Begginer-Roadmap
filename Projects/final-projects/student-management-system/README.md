# Capstone Track 1: Student Management System

## 📌 Project Description
Build a comprehensive academic administration application supporting complete student lifecycle management, course enrollment tracking, JSON persistence, and CSV academic roster exporting.

## 🎯 Requirements
- Full CRUD operations (Add, View, Search, Update, Delete)
- Fictional student records (`STU-001`, Name, Age, Department, Email, Courses)
- Automated Unit Tests verifying data model and service controller logic
