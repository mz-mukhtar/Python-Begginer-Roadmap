# 📘 Week 8 Overview: APIs, Testing, and Capstone Development

Welcome to **Week 8** — the final week of the Python Beginner to Practical Developer Roadmap! 🎉🚀

This capstone week brings together all your foundational knowledge, Object-Oriented architecture, data handling, and development tools. You will learn how to connect your Python applications to external live web services using **REST APIs**, write automated software verification tests using `unittest`, and build, test, document, and present your comprehensive **Capstone Final Project**.

---

## 🎯 Learning Objectives

By completing Week 8, you will be able to:
- Explain REST API concepts (HTTP methods, endpoints, status codes, JSON payloads)
- Fetch live data from web endpoints using the Python `requests` library
- Distinguish between manual verification and automated unit testing
- Write clean automated assertions using Python's built-in `unittest` framework
- Architect, build, debug, and document a complete capstone software project
- Deliver an effective technical presentation demonstrating your final application
