# 📘 Week 4 Overview: Strings and Data Persistence

Welcome to Week 4! This week bridges the gap between temporary memory and permanent disk storage. You will dive deep into text manipulation with string methods and f-strings, learn how to open, read, and write plain text files, and serialize structured application data using JSON and CSV files.

---

## 🎯 Learning Objectives

By completing Week 4, you will be able to:
- Manipulate, slice, format, and search strings using built-in methods and f-strings
- Validate text input using `isdigit()`, `isalpha()`, and `isalnum()`
- Read and write text files safely using context managers (`with open()`)
- Work with relative and absolute file paths (`pathlib`)
- Save and load structured JSON objects and arrays (`json.dump`, `json.load`)
- Read and write tabular CSV files
- Build persistent applications that save state between runs

---
