# 🚀 Session Challenge

## Personal Profile Card

### Description

Build a structured Personal Profile Card that displays student details inside a visually formatted text box.

### Requirements

- Store the student's name, age, school, department, and favorite technology in variables
- Display a decorative header and footer using symbols like `=` or `-`
- Print each detail clearly labeled

### Expected Features

- Clean visual borders
- Accurate variable usage
- Readable output formatting

### Hints

1. Define your variables at the top of the file before calling `print()`.
2. Use string multiplication like `"=" * 30` or print literal separator bars.

### Optional Improvements

- Add an extra variable for current learning goal
- Format labels to align neatly

Try to complete the challenge before opening the solution.
