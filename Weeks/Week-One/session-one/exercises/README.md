# 🧑💻 Session Exercises

Complete these exercises after reading the lesson and running the examples.

---

## Exercise 1 — Print a Welcome Message

### Goal

Practice using the `print()` function to display text on the screen.

### Requirements

- Display the message `"Hello, Python Learner!"`
- Display a second line saying `"Ready to start coding!"`

### Example Output

```text
Hello, Python Learner!
Ready to start coding!
```

### Starter File

Open:

`starter/exercise-01.py`

### Hint

Use two separate `print()` function calls on separate lines.

---

## Exercise 2 — Display Personal Information

### Goal

Practice printing organized personal details.

### Requirements

- Print your name on line 1
- Print your target career role on line 2
- Print your country on line 3

### Example Output

```text
Name: Abel
Role: Software Developer
Country: Ethiopia
```

### Starter File

Open:

`starter/exercise-02.py`

### Hint

Pass text inside double quotes `"..."` inside `print()`.

---

## Exercise 3 — Fix Unreadable Code

### Goal

Identify and fix syntax errors in poorly written print statements.

### Requirements

- Fix missing quotation marks
- Fix missing closing parentheses
- Ensure all three lines execute cleanly

### Example Output

```text
Python is fun!
Learning step by step.
Code works cleanly!
```

### Starter File

Open:

`starter/exercise-03.py`

### Hint

Check each `print(...)` statement carefully for matching quotes and parentheses.

---

## Exercise 4 — Add Useful Comments

### Goal

Practice documenting your Python code using comments.

### Requirements

- Add a header comment explaining what the script does
- Add inline comments above each print statement
- Ensure the code runs without error

### Example Output

```text
Student Profile Script Active
Welcome!
```

### Starter File

Open:

`starter/exercise-04.py`

### Hint

Start comments with the `#` symbol.
