# 🚀 Session Challenge

## Scholarship Eligibility System & Smart Student Grade System

### Description

Build a complete Smart Student Grade System that evaluates exam score and attendance to determine both letter grade and scholarship eligibility.

### Requirements

- Collect student name, score (0-100), and attendance percentage (0-100)
- Validate inputs are within valid range
- Calculate letter grade A, B, C, F
- Check scholarship eligibility (Score >= 85 AND Attendance >= 90)
- Display structured final report

### Expected Features

- Range validation
- Combined logical conditions (`and` / `or`)
- Clean report card output

Try to complete the challenge before opening the solution.
