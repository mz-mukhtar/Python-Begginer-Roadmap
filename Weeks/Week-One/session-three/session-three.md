# 📘 Week 1 – Session Three: Conditions and Weekly Project

## 📌 Session Overview

This session focuses on practical application and decision-making logic in Python. Students will practice writing conditional branches (`if`, `elif`, `else`), combining comparisons with logical operators (`and`, `or`, `not`), handling nested conditions, and completing their first guided weekly project: the **Student Grade System**.

## 🎯 Learning Objectives

Students will be able to:
- Use `if`, `elif`, and `else` statements to control program flow
- Combine multiple conditions cleanly using logical operators (`and`, `or`, `not`)
- Implement nested conditional blocks for complex decision logic
- Build, test, and debug an interactive **Student Grade System** application

## 📚 Planned Topics

- `if`
- `elif`
- `else`
- Logical operators
- `and`

- Nested conditions
- Grade System project
- Weekly exercises

## 🛠️ Planned Weekly Project

**Student Grade System**: An interactive program that accepts student scores, validates input ranges, determines letter grades and honor distinctions using conditional branches, and displays formatted feedback.
