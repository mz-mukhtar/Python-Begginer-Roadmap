# 🚀 Session Challenge

## Student Result Calculator

### Description

Build an interactive Student Result Calculator that collects scores for three subjects, computes the total and percentage average, and prints a formatted report card.

### Requirements

- Collect student name via `input()`
- Collect scores for Math, Science, and English via `input()` converted to floats
- Calculate total score out of 300
- Calculate percentage average out of 100
- Display formatted summary report

### Expected Features

- Input type conversion
- Arithmetic calculations
- Clean terminal report formatting

### Hints

1. Convert inputs immediately using `float(input(...))`.
2. Compute average as `total / 3`.

Try to complete the challenge before opening the solution.
