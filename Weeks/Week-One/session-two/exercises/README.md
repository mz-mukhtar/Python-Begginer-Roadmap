# 🧑💻 Session Exercises

Complete these exercises after reading the lesson and running the examples.

---

## Exercise 1 — Personal Information Program

### Goal

Practice using variables to store and display personal details.

### Requirements

- Create variables for first name, last name, and city
- Print a sentence combining these variables

### Example Output

```text
First Name: Abel
Last Name: Tadesse
City: Addis Ababa
```

### Starter File

Open:

`starter/exercise-01.py`

### Hint

Assign each value to a clearly named variable before printing.

## Exercise 2 — Age Calculator

### Goal

Practice converting string input into integers using `int()`.

### Requirements

- Ask the user for their birth year
- Calculate their age assuming the current year is 2026
- Display their calculated age

### Example Output

```text
Enter birth year: 2005
Your calculated age is: 21
```

### Starter File

Open:

`starter/exercise-02.py`

### Hint

Remember that `input()` always returns a string; wrap it with `int()`.

## Exercise 4 — Temperature Converter

### Goal

Convert Celsius temperature to Fahrenheit.

### Requirements

- Given Celsius value `25`
- Calculate Fahrenheit using `(celsius * 9/5) + 32`
- Print the result clearly

### Example Output

```text
25 Celsius is equal to 77.0 Fahrenheit.
```

### Starter File

Open:

`starter/exercise-04.py`

### Hint

Follow correct order of operations in your formula.

---

## Exercise 5 — Subject Average Calculator

### Goal

Calculate the average of three exam scores.

### Requirements

- Store three scores: `80`, `90`, and `85`
- Calculate the total sum
- Calculate the average by dividing total by `3`
- Display both total and average

### Example Output

```text
Total Score: 255
Average Score: 85.0
```

### Starter File

Open:

`starter/exercise-05.py`

### Hint

Use parentheses `(s1 + s2 + s3) / 3` to compute average correctly.
