# 📘 Week 7 – Session One: Virtual Environments and Packages

## 📌 Session Overview

This session introduces third-party Python packages and dependency management. Students will learn how to create isolated Python virtual environments (`venv`) so each project has its own dedicated dependencies without conflicting with system packages.

## 🎯 Learning Objectives

Students will be able to:
- Explain what Python packages are and how they extend standard functionality
- Create, activate, and deactivate Python virtual environments using `venv`
- Install and inspect external Python libraries using `pip`
- Export and reproduce project environments using `requirements.txt`

## 📚 Planned Topics

- Python packages
- `pip`
- Virtual environments
- Creating a virtual environment
- Activating virtual environments
- Installing packages
- Viewing packages
- `requirements.txt`

## 🛠️ Planned Practical Activity

**Virtual Environment Setup**: Students will create an isolated virtual environment, install third-party libraries using `pip`, and generate a reproducible `requirements.txt` file for a Python project.
