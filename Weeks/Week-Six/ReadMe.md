# 📘 Week 6 Overview: Object-Oriented Programming

Welcome to Week 6! 🎉

Over the past five weeks, you have progressed from writing simple scripts to building file-backed, error-safe applications. This week, you step into professional software engineering by learning **Object-Oriented Programming (OOP)**.

In OOP, you organize your code around **objects**—real-world models that bundle data and actions together. You will learn how to design **Classes** (`class`), create **Objects**, initialize attributes using `__init__()`, write class **Methods**, manage object collections, and implement basic inheritance.

---

## 🎯 Learning Objectives

By completing Week 6, you will be able to:
- Explain what Object-Oriented Programming is and why professional developers use it
- Design classes and create distinct object instances
- Initialize object attributes using the `__init__()` constructor and `self` keyword
- Write class methods to encapsulate object behavior
- Manage collections of objects inside application controllers
- Implement basic inheritance between parent and child classes
- Architect complete Object-Oriented Python applications
