# 📘 Week 6 Overview: Object-Oriented Programming & Final Project

Welcome to the final week of the Python Beginner Roadmap! 🎉

Over the past five weeks, you have progressed from writing one-line `print()` statements to building file-backed, error-safe applications. This week, you will take the final step into professional software engineering by learning **Object-Oriented Programming (OOP)**.

In OOP, you organize your code around **objects**—real-world models that bundle data and actions together. You will learn how to design **Classes** (`class`), create **Objects**, initialize attributes using `__init__()`, and write class **Methods**. Finally, you will apply everything you have learned across all six weeks to build, test, polish, and present your **Capstone Final Project**!

---

## 🏛️ Session 1: Introduction to Object-Oriented Programming (OOP)

### ✅ What You’ll Learn:
- What Object-Oriented Programming is and why professional developers use it
- The difference between a **Class** (blueprint) and an **Object** (instance)
- Using the `__init__()` constructor method and the `self` keyword
- Creating multiple distinct objects from a single class
- Adding class **Methods** to perform actions on object data
- How to convert a function-based application into a clean class-based application

### 🛠️ Mini Project:
- **Class-Based Student Model**: Creating reusable `Student` objects with custom display methods.

---

## 🎓 Session 2: Final Project Development & Presentation

### ✅ What You’ll Learn:
- Reviewing the complete 6-week roadmap (Variables $\rightarrow$ Conditions $\rightarrow$ Loops $\rightarrow$ Functions $\rightarrow$ Files $\rightarrow$ Errors $\rightarrow$ OOP)
- Structuring a complete Python project folder with `main.py`, class files, JSON storage, and `README.md`
- Building an integrated capstone project from the **Projects Manual**
- Testing techniques (valid inputs, invalid inputs, edge cases) and practical debugging skills
- Writing professional project documentation
- Preparing and delivering your final project demo presentation!

---

## 🏆 Expected Student Outcomes:
By completing Week 6, you will be able to:
- Write organized Object-Oriented Python code using classes and methods
- Architect complete, reliable CLI applications from start to finish
- Debug, test, and document Python projects independently
- Confidently present technical software projects
