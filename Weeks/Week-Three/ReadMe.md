# 📘 Week 3 Overview: Functions and Program Organization

Welcome to Week 3! As your scripts grow larger, writing everything in one long file becomes difficult to read and maintain. This week, you will learn how to organize code using **Functions** (`def`), return values, manage variable scope, and decompose programs into modular components.

---

## 🎯 Learning Objectives

By completing Week 3, you will be able to:
- Define and call reusable Python functions using `def`
- Pass data into functions using positional and keyword arguments
- Return computed values from functions using `return`
- Explain the difference between local and global variable scope
- Break large programs into clean, single-responsibility functions
- Build interactive CLI menu applications

---

- **Summary**: Learn what functions are, why they are useful, how to define (`def`) and call functions, and how to work with parameters and arguments.
