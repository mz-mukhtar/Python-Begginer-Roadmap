# 📘 Week 2 Overview: Loops and Python Collections

Welcome to Week 2! This week, you will explore **repetition and data collections** in Python. You will learn how to repeat tasks using loops (`while`, `for`), store multiple values using lists and tuples, and manage key-value pairs and unique collections using dictionaries and sets.

---

## 🎯 Learning Objectives

By completing Week 2, you will be able to:
- Repeat tasks and automate logic using `while` and `for` loops
- Control loop execution using `break` and `continue`
- Store, index, update, and slice ordered sequences using lists and tuples
- Store structured key-value profiles using dictionaries
- Eliminate duplicate entries automatically using sets
- Build interactive data collector applications

---

## 🔁 Session 1: Loops
- **Summary**: Master `while` loops, `for` loops, `range()`, loop counters, infinite loops, `break`, `continue`, and simple nested loops.
- **Status**: Complete

---

## 🧺 Session 2: Lists and Tuples
- **Summary**: Create, index, slice, and modify lists (`append()`, `remove()`), loop through lists, and understand immutable tuples vs. mutable lists.
- **Status**: Complete

---

## 🗝️ Session 3: Dictionaries, Sets, and Weekly Project
- **Summary**: Work with key-value pairs in dictionaries, add/update/remove dictionary items, handle nested data, use sets for unique values, and build the **Student Information Collector** project.
- **Status**: Complete

---

## 🛠️ Weekly Project

**Student Information Collector**: An interactive program that collects multi-field student profiles into structured dictionaries, prevents duplicate entries using sets, and displays organized student lists.

---

## 🏆 Expected Student Outcomes
By the end of Week 2, students will be able to automate repetitive workflows and structure multi-item datasets using lists, tuples, dictionaries, and sets.
