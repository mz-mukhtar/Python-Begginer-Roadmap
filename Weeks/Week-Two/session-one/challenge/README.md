# 🚀 Session Challenge

## Number Guessing Logic

### Description

Build a Number Guessing Logic game using a fixed secret number (e.g. 25). Do not use the `random` module because it has not been introduced yet.

### Requirements

- Secret number is 25
- Loop repeatedly asking user for their guess
- If guess < 25, print `"Too low!"`
- If guess > 25, print `"Too high!"`
- When correct, print victory message and stop loop

### Expected Features

- `while True` loop with `break`
- Conditional feedback

Try to complete the challenge before opening the solution.
