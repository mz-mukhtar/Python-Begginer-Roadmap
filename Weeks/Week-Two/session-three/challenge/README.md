# 🚀 Session Challenge

## Contact Information Collector & Student Information Collector

### Description

Build the capstone Student Information Collector that collects multiple student dictionary profiles inside a list, tracks unique departments using a set, and allows searching by name.

### Requirements

- Collect student name, age, and department
- Store each student as a dictionary inside a list
- Store department inside a set
- Display all profiles and unique departments
- Search for student by name

Try to complete the challenge before opening the solution.
