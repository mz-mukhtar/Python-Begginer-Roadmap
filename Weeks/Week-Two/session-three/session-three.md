# 📘 Week 2 – Session Three: Dictionaries, Sets, and Student Information Project

## 📌 Session Overview

This session introduces key-value data modeling with dictionaries and unique value filtering with sets. Students will learn how to store, update, and inspect structured student profiles and apply these collection types by building the **Student Information Collector** project.

## 🎯 Learning Objectives

Students will be able to:
- Create and manipulate dictionaries using key-value pairs
- Add, update, and remove items from dictionaries safely
- Iterate through dictionary keys, values, and items using built-in methods
- Structure nested data sets representing multi-field records
- Use sets to filter out duplicate values automatically
- Implement the interactive **Student Information Collector** weekly project

## 📚 Planned Topics

- Dictionaries
- Keys
- Values
- Adding dictionary values
- Updating values
- Removing values
- Dictionary methods
- Nested data
- Sets
- Unique values
- Student Information Collector project

## 🛠️ Planned Weekly Project

**Student Information Collector**: An interactive application that gathers multi-field student profiles (ID, name, courses, and attributes) into structured dictionaries, prevents duplicate entries using sets, and formats the collected data cleanly.
